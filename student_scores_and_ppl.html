df <- df %>%

Perplexity per Discourse Type

Discourse elements should differ in perplexity. High-entropy elements like “Data” vary more than low-entropy elements like “Primary_Claim”s, since there should only be two types of primary claims (“I agree” or “I disagree”), but there are countless potential examples.

ggplot(df_annot,
    aes(y = ppl, x = adjudicated_discourse_type,
    fill = adjudicated_discourse_type)) +
geom_boxplot() +
theme(legend.position = "none") +
xlab("Discourse Type") +
ylab("Perplexity")
